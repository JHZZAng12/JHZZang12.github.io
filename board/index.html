Supabase Setup

처음 한 번만 필요한 테이블 준비

아직 테이블을 만들지 않았다면 Supabase SQL Editor에서 supabase/board_posts.sql 내용을 실행하면 됩니다.

1. SQL 실행

supabase/board_posts.sql 파일을 Supabase SQL Editor에 붙여 넣고 실행합니다.

2. 정책 확인

익명 사용자도 읽기와 쓰기가 가능하도록 RLS 정책이 같이 생성됩니다.

3. 배포 반영

이 저장소를 GitHub Pages에 푸시하면 별도 서버 없이 게시판이 바로 열립니다.
